# EVM-Scratch
EVMをスクラッチで作るリポジトリです。

## 動かし方

- 仮想環境の作成

    ```bash
    python -m venv myenv
    ```

- 仮想環境の有効

    ```bash
    source myenv/bin/activate
    ```
